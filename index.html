What's a framework?

  • Not a set of libraries
  • Not a set of dependencies
  • A set of independencies
  • A set of guidelines
  • A set of best practices

Making the interwebs is easy

  1. User makes request
  2. Inspect incoming data (URL, cookies, whatnot)
  3. Validate incoming data
  4. Decide which other data needs to be fetched
  5. Transform fetched data into something reasonable (e.g. HTML)
  6. Spit the data back out

Trivial.

Making web 2.0 is easy

  1. User interacts with something
  2. Fetch some data
  3. Transform fetched data into something reasonable (e.g. innerHTML)

Trivial.
